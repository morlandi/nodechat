nodechat
========

A simple chat service based on Node.js & WebSocket, used to learn Node.js

Adapted from:

https://gist.github.com/martinsik/2031681

Usage (in localhost)
--------------------

::

    npm install
    node chat-server.js

then open **frontend.html** in your browser
